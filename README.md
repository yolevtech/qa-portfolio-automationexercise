# QA Portfolio — Automation Exercise

Este repositório contém um projeto prático completo de **Quality Assurance** utilizando a aplicação web [Automation Exercise](https://automationexercise.com).  
O objetivo é demonstrar, de forma prática, o conhecimento no processo de testes de software, desde o planejamento até a automação.

---

## 🎯 Objetivo do Projeto

Simular o trabalho real de um QA em um produto web, aplicando atividades comuns do dia a dia:

- Análise da funcionalidade
- Criação de plano de testes
- Elaboração de casos de teste
- Execução de testes manuais
- Registro de evidências
- Reporte de defeitos
- Automação de testes
- Escrita de cenários BDD

---

## 🧪 Funcionalidades Testadas

- Login de usuário
- Logout
- Validação de campos
- Tratamento de erros
- Comportamento de sessão

## 🗂 Estrutura do Repositório

| Pasta | Conteúdo |
|-------|----------|
| `/test-plan` | Plano de testes em Markdown |
| `/test-cases` | Doc de casos de testes + planilha em Markdown |

| `/bug-reports` | Registro de defeitos encontrados |
